如何加入星落小镇？

1

准备游戏客户端

请确保你使用的是 Minecraft Java 版 1.21.11。推荐使用 HMCL 或 PCL2 启动器。

2

添加服务器

在多人游戏界面，点击“添加服务器”，在服务器地址栏输入：play.xl-town.com
